# Aesthetic Routing

The cover's visual register is chosen BY THE SUBJECT — the niche, the mood, the topic —
automatically. The user does not name a style. A 国学/古诗词 cover routes to ink-wash; a
productivity cover routes to a Swiss grid; a skincare cover routes to bold editorial. Route
first, then the title-legibility harness runs inside that register.

This replaces "pick a master by hand." The skill infers the register and pulls the tokens.

## Route by niche / mood → register

| Cover topic / mood | Register | Tokens to apply | Title treatment |
| --- | --- | --- | --- |
| 知识 / 干货 / 效率 / 科技 / 工具 | 瑞士网格 (Müller-Brockmann / Vignelli) | a real grid, one accent on neutral ground, one strong sans flush-left, generous whitespace | heavy sans, snapped to the grid |
| 国风 / 古风 / 国学 / 诗词 / 传统文化 | 水墨 / 文人画 | rice-paper ground, ink wash, 留白, vertical 书法 title, a red 印章 accent | vertical brush calligraphy, one seal |
| 产品 / 好物 / 电商 / 测评 | 工作室静物 (Irving Penn / Hockney) | clean neutral seamless or flat sun palette, one soft side light, one material texture; composite the user's REAL product photo | small tag in a corner; the object leads |
| 情绪 / 共鸣 / 治愈 / emo / 个人随笔 | 斜射光 + 留白 (Hopper + Kenya Hara) | low directional light, long soft shadows, muted palette + one warm accent, lots of empty space | quiet, lower-contrast, honest |
| 美食 | 暖调编辑 (warm Hockney) | warm appetizing palette, clean planes, one hero dish, soft daylight | small warm tag |
| 穿搭 / 时尚 / 美妆 | 大胆极简编辑 (Malika Favre) | two saturated flats + negative space, flat banded light, magazine-editorial crop | bold, minimal, off to one side |
| 母婴 / 萌宠 / 可爱 | 圆润流行 (Olimpia Zagnoli) | soft rounded silhouettes, a tested high-saturation palette, flat and buoyant | rounded, friendly weight |

| 旅行 / 风景 / 城市 | 电影感 (Hopper / Tom Haugomat) | cinematic wide framing, directional light, limited palette, scale + atmosphere | small, lets the scene lead |

## How to route

1. Read the topic + the account's niche + the mood.
2. Pick the register from the table. When two fit, pick the one a follower of this niche
   expects to see (国学 belongs on rice paper, not a neon grid).
2.5. If the topic matches no row, route to the editorial-grid default above — NEVER to a generic
   gradient. If it spans two rows, pick the register a follower of the account's PRIMARY niche
   expects (a 国学读书号 → 水墨 even if the post is a 清单; a 效率号 doing a 国风选题 → still its
   grid). State the routed register and the one-line reason in the plan so the choice is auditable.
3. Apply that register's tokens to the whole cover — ground, palette, light, type voice.
4. The title-legibility harness (one promise, thumbnail-readable hero line, restraint) runs
   INSIDE the register, unchanged.
5. The user may override; the default is the routed register, never a generic gradient.

## Compatibility rule

The host (a cover) forbids clutter. Only route to registers whose restraint fits: never apply
a maximalist token (Yokoo collage, Klimt ornament) to a clean cover. Scher "type-as-image" is
allowed only when the topic is genuinely loud (活动/潮流) and the title stays readable.

## The ink register (国风/国学 covers, often missed)

For 古诗词 / 国学 / 传统文化 / 国风 topics, default to: warm rice-paper ground, an ink-wash or
fine-line motif, generous 留白, the title set as vertical brush 书法, and a single red 印章 as
the one warm accent. Black ink + paper + one seal-red — no gradients, no neon. See the
East-Asian section of the master library for the transferable tokens.
